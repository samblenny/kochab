Kochab

Demo Pages

  • /nasa-apod.html — Shows an image and caption for NASA's Astronomy Picture of the Day from the NASA API. This uses a serverless function to request JSON from NASA and transform it into an HTML fragment.

Demo API Endpoints

  • /api/hello — Simple serverless function to demonstrate the site is working. This does not rely on accessing any third party APIs. The content-type is application/json.
  • /api/nasa-apod — This requests JSON from NASA's API, then transforms it into an html fragment. It's a little silly to do this server-side, but the point was to experiment with the plumbing for handling API keys and transforming data from a third party API. The API key comes from an environment variable, configured either through the dashboard in Vercel, or manually for local dev. The content type is text/html.

Environment Variables

To run this locally, the dev server needs an environment variable to be set for the API key. For more information, refer to the README in the samblenny/kochab repository on GitHub.
